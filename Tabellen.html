Tabellen:

Tabellen werden auch mit Tags geschrieben aber mann muss für die Tabelle selber und für
die einzelnen Punkte jeweils ein Tag geschrieben werden. Um die Tabelle selber anzulegen
braucht man den Tag <ol>(für eine geordnete Tabelle) oder <ul> (für eine ungeordnete
Tabelle). Für die einzelnen Punkte brauch man den Tag <li>, diesen kann man auf jegliche
Tabellen anwenden.Um mit der Tabelle aufzuhören muss mann den Tag der Tabelle oder der
einzelnen Punkte nur mit einem "/"vor dem Inhalt des Tags setzen:

geordnete Tabelle:

  1. (erstens)
  2. (zweitens)
  3. (drittens)
  4. ...

ungeordnete Tabelle:

  • Beispiel 1
  • Beispiel1
  • ...
